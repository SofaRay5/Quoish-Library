#固体物理 
在[[晶体]]这一章中，讲到晶体中的原子是严格按照一定的规律排列的，各种原子都严格地位于原胞中的相应位置，而原胞又严格地排列在规则的格点位置，即原子排列具有严格的周期性。
这实际上只是一种理想模型，而实际的晶体在形成时，常常会遇到一些不可避免的干扰，造成实际晶体与理想晶体的一些差异。即现实存在的晶体的原子排列并不像理想的那样完美无缺，而是存在着各种各样对周期性的偏离。
我们把这些对理想周期结构的偏离称为*缺陷*。
缺陷按照几何形状可以概括为四种类型：
- *点缺陷*：是发生在晶体中一个或几个晶格常数范围内的一种缺陷，如晶体中空格点、外来的杂质原子；
- *线缺陷*：是发生在晶体中一条线周围的一种缺陷，如位错；
- *面缺陷*：是发生在晶体中的二维缺陷，如界面；
- *体缺陷*：是发生在晶体中的三维缺陷，如晶体中的包裹体。
晶体中形形色色的缺陷与晶体的物理、化学性质有密切关系，许多材料特性对其中的缺陷很敏感，因此控制材料缺陷（包括杂质）是十分重要的课题。
# 点缺陷
点缺陷是晶体中以空位、间隙原子、杂质原子为中心，在一个或几个晶格常数的微观区域内，晶格结构偏离严格周期性而形成的畸变区域。
由于热振动，某些原子脱离了格点，就会形成*空位*。
若有原子进入晶格中的间隙位置，就会形成*填隙原子*。
空位和填隙原子使得晶格周期性遭到破坏，但这种破坏只发生在几个晶格常数范围内，这是点缺陷的主要来源。
## Frenkel缺陷
由于热涨落，晶体格点上的原子可能获得一定动能，脱离正常格点位置而进入格点间隙位置，从而形成填隙原子，同时在原来的格点位置上留下空位，这样一来晶体中将存在等浓度的空位和填隙原子。
这种**空位-间隙原子对**称为*Frenkel缺陷*。
## Schottky缺陷
由于热涨落，晶体格点上的原子可能获得一定动能，以至于**从晶体内部而迁移到晶体表面**上的某一格点位置，在晶体表面上构成新的一层，从而在晶体内部原来的格点位置上留下**空位**，这种缺陷称为*Schottky缺陷*。
晶体中Schottky缺陷产生的方式可以是不同的。
晶体邻近表面的原子可以由于热涨落跳到表面，从而产生一个空位，附近原子跳到这个空位上，就又产生一个新空位，这样空位可以逐步跳跃到晶体内部。
也可能由于热涨落晶体内部原子脱离格点，产生一个空位，这个原子可以经过多次跳跃，而跑到晶体表面的正常格点位置上，在晶体内形成空位。
## 反Schottky缺陷
与Schottky缺陷相对应的，还有一种*反Schottky缺陷*，也称为*间隙原子缺陷*。它是**晶体表面某一格点的原子移到晶体内部的间隙位置**。
显然，从晶体表面挤入拥挤的晶体内部形成反Schottky缺陷需要更大的能量，所以除小半径杂质原子外，一般不易单独形成此种缺陷。
![[点缺陷.png]]

|    特征    |    Frenkel缺陷     |   Schottky缺陷   |  反Schottky缺陷   |
| :------: | :--------------: | :------------: | :------------: |
| **缺陷组成** |    空位 + 间隙原子     |       空位       |       间隙       |
| **原子来去** |   晶格位置 → 间隙位置    |  晶格位置 → 晶体表面   |  晶体表面 → 间隙位置   |
| **晶体体积** |       基本不变       |       增加       |   基本不变或略微减小    |
| **晶体密度** |       基本不变       |       降低       |    升高或基本不变     |
| **电荷效应** | 空位与间隙原子电荷相反，整体中性 | 阴阳离子空位等量，整体中性  | 阴阳离子间隙等量，整体中性  |
| **常见性**  | 常见（尤其在阳离子小的晶体中）  | 非常常见（尤其在离子晶体中） |      极其罕见      |
| **主导因素** |  阳离子尺寸小，易于进入间隙   | 阴阳离子尺寸相近，结构开放  | 需要极高的形成能，通常不考虑 |
Frenkel缺陷和Schottky缺陷都可以由热运动产生，所以也称为*热缺陷*（本征缺陷）。
由于热运动的随机性，缺陷也可以消失，称为*复合*。
一定温度下，缺陷的产生和复合过程是相互平衡的，缺陷将保持一定平衡浓度。
## 杂质原子缺陷
实际晶体中总是存在某些微量杂质。
杂质的来源一方面是在晶体生长过程中自然引入的，另一方面是为了改善晶体性质被人为引入的。
杂质原子在晶体中的占据方式有两种：
- 一种是杂质原子占据基质原子的位置，称为*替位式杂质缺陷*；
- 一种是杂质原子进入基质原子间的间隙位置，称为*填隙式杂质缺陷*。

下图表示硅晶体中填隙式杂质和替位式杂质的示意图，图中$A$为间隙式杂质，$B$为替位式杂质。![[硅晶体中的间隙式杂质和替位式杂质.png]]
如果杂质原子相较于基质原子的原子大小非常小，那一般会形成填隙式杂质。
如果杂质原子相较于基质原子的原子大小比较接近、电负性比较接近，那一般会形成替位式杂质。
向晶体中掺入杂质原子有多种方式，如晶体生长时进行高温扩散或离子注入等。
## 色心
*色心*（Color Center）是晶体中的一种点缺陷，它能选择性地吸收特定波长的可见光，从而使原本无色或白色的晶体呈现出颜色。
### F心
$\mathrm{F}$心是最典型的一种色心，其名字来自德语“Farbenzentrum”（颜色中心）。
*$\mathrm{F}$心*指的是：**一个阴离子空位** + **一个被该空位静电势阱所捕获的电子**。
其形成过程如下：
1. 首先，晶体中需要产生一个阴离子空位。这通常通过Schottky缺陷（既会产生阴离子空位也会产生阳离子空位）的形成来实现。
2. 然后，这个阴离子空位本身像一个正电荷中心，它会吸引并捕获一个自由的电子。
3. 这个被捕获的电子在空位处具有一系列特定的量子化能级，能够吸收可见光。

以晶体氯化钠举例，纯净的氯化钠本来是无色的，但如果将其放在钠蒸汽中骤冷就会因产生色心而呈现黄色（岩盐）。因为高温，氯化钠晶体中会因为热运动产生很多Schottky缺陷（等量的$\mathrm{Na^{+},Cl^{-}}$的空位）；而钠蒸汽在晶体表面发生电离，产生一个$\mathrm{Na^+}$和一个自由电子$\mathrm{e}^-$。从晶体内部冒到晶体表面的$\mathrm{Cl^{-}}$和$\mathrm{Na^+}$结合占据一个新的格点；而自由电子潜入到晶体内部，被呈正电的$\mathrm{Cl^-}$空位所吸引，被固定在空位附近运动，这就类似于一个“氢原子”。骤冷，这样的结构就被固定下来了。
**形成色心的过程会改变原先晶体的化学计量比**。比如在上述岩盐形成的过程中，$\mathrm{Na}$原子的含量变多了，钠和氯就不再是1:1了。
类似的$\mathrm{KCl}$在$\mathrm{K}$蒸汽中会变成紫色，$\mathrm{LiF}$在$\mathrm{Li}$蒸汽中会变成粉红色，$\mathrm{CaF_{2}}$在$\mathrm{Ca}$蒸汽中会变成紫色。
### V心
$\mathrm{V}$心是与F心相对的一种色心，其名字来自“Vacancy”。
*$\mathrm{V}$心*指的是：**一个阳离子空位** + **一个被该空位静电势阱所捕获的空穴**。
其形成过程如下：
1. 首先，晶体中需要产生一个阳离子空位。这通常通过Schottky缺陷（既会产生阴离子空位也会产生阴离子空位）的形成来实现。
2. 然后，这个阳离子空位本身像一个负电荷中心，它会吸引并捕获一个自由的空穴。
3. 这个被捕获的空穴在空位处具有一系列特定的量子化能级，能够吸收可见光。

以晶体氯化钠举例，纯净的氯化钠本来是无色的，但如果将其放在过量氯气中，就会因产生色心而呈现其他颜色，但是$\mathrm{V}$心通常没有$\mathrm{F}$心稳定。因为高温，氯化钠晶体中会因为热运动产生很多Schottky缺陷（等量的$\mathrm{Na^{+},Cl^{-}}$的空位）；而氯气$\mathrm{Cl_{2}}$在晶体表面抢走了表面$\mathrm{Cl^-}$的两个电子，鸠占鹊巢，迫使原来的两个$\mathrm{Cl^-}$变成$\mathrm{Cl}$，这个$\mathrm{Cl}$就是“空穴”，等效地带着正电荷。从晶体内部冒到晶体表面的$\mathrm{Na^+}$和新来的$\mathrm{Cl^-}$结合占据一个新的格点；而空穴也就是$\mathrm{Cl}$潜入到晶体内部，被呈负电的$\mathrm{Na^+}$空位所吸引，被固定在空位附近运动。这时，一种图景是视$\mathrm{Cl}$与周围的$\mathrm{Cl^-}$不断交换着电子，另一种图景是$\mathrm{Cl}$会和周围的$\mathrm{Cl^-}$复合为二聚体$\mathrm{Cl_{2}^-}$。

# 线缺陷
线缺陷是晶体内部偏离周期性点阵的一维缺陷。
晶体中最重要的一种线缺陷是*位错*。位错既可以说是晶体结构中的一种缺陷，也可以说是原子排列中的一种特殊组态。
位错在晶体的范性形变（遭到应力时发生的永久性形变）、强度、断裂、相变以及它的结构敏感性问题中起着重要作用。
## 位错理论的提出
最初，人们认为范性形变造成晶面的滑移是晶面之间整体的，刚性的相对滑移。
1926年，Frenkel在运用上述思想估算临界应切力时，发现实际的弹性极限远小于理论估计的值。这意味着滑移有可能并不是晶面之间整体性的运动，而是原子相继运动的结果，有些原子在其他原子还没有运动之前就开始运动了。
这就导致了位错理论的提出。
## 位错的基本类型
位错最基本的类型有刃形位错和螺形位错。
### 刃形位错
![[刃形位错.jpg]]
如图所示，假如沿滑移面$ABCD$切开晶体，然后让滑移面两侧的晶体沿矢量$b$错位一个原子间距，这就导致在晶体内部多出了一个半晶面。这个半晶面在晶体内的边缘类似一个刀刃，这个刀刃就是晶体滑移区与未滑移区的交线，称为*位错线*。且在此线上附近的原子发生严重错排，形成线缺陷——位错，故称此位错为*刃形位错*。
- 刃形位错有一个额外的半原子面。
- 位错线可以是直线、折线或曲线，垂直于滑移方向，但**不能在晶体内中断**。
- 晶体中存在刃形位错后，位错周围的点阵发生弹性畸变，上方受压，下方受拉。
- 位错线周围的畸变区，每个原子有较大的平均能量，畸变区是一个狭长的管道。
### 螺形位错
![[螺形位错.jpg]]
如图，如果让滑移面两侧的晶体沿矢量$b$错位一个原子间距，则围绕滑移区与未滑移区的交线，错排原子形成一矩形螺线，称此缺陷为*螺形错位*。
- 刃形位错有多余的半个原子面，螺形位错没有。
- 刃形位错的位错线垂直于滑移矢量$b$，螺形位错的位错线平行于滑移矢量$b$。
- $b$又称Burgers矢量，其模垂直于滑移方向上的平衡原子间距，其方向代表滑移方向。
### 混合位错
除了刃形位错和螺形位错，还有位错线与滑移矢量既不平行又不垂直的混合型位错，位错线和滑移方向可以成任意角度。
