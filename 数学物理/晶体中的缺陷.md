#固体物理 
在[[晶体]]这一章中，讲到晶体中的原子是严格按照一定的规律排列的，各种原子都严格地位于原胞中的相应位置，而原胞又严格地排列在规则的格点位置，即原子排列具有严格的周期性。
这实际上只是一种理想模型，而实际的晶体在形成时，常常会遇到一些不可避免的干扰，造成实际晶体与理想晶体的一些差异。即现实存在的晶体的原子排列并不像理想的那样完美无缺，而是存在着各种各样对周期性的偏离。
我们把这些对理想周期结构的偏离称为*缺陷*。
缺陷按照几何形状可以概括为四种类型：
- *点缺陷*：是发生在晶体中一个或几个晶格常数范围内的一种缺陷，如晶体中空格点、外来的杂质原子；
- *线缺陷*：是发生在晶体中一条线周围的一种缺陷，如位错；
- *面缺陷*：是发生在晶体中的二维缺陷，如界面；
- *体缺陷*：是发生在晶体中的三维缺陷，如晶体中的包裹体。
晶体中形形色色的缺陷与晶体的物理、化学性质有密切关系，许多材料特性对其中的缺陷很敏感，因此控制材料缺陷（包括杂质）是十分重要的课题。
# 点缺陷
## Frenkel缺陷
由于热涨落，晶体格点上的原子可能获得一定动能，脱离正常格点位置而进入格点间隙位置，从而形成**填隙原子**，同时在原来的格点位置上留下**空位**，这样一来晶体中将存在等浓度的空位和填隙原子。
这种**空位-间隙原子对**称为*Frenkel缺陷*。
## Schottky缺陷
由于热涨落，晶体格点上的原子可能获得一定动能，以至于**从晶体内部而迁移到晶体表面**上的某一格点位置，在晶体表面上构成新的一层，从而在晶体内部原来的格点位置上留下**空位**，这种缺陷称为*Schottky缺陷*。
晶体中Schottky缺陷产生的方式可以是不同的。
晶体邻近表面的原子可以由于热涨落跳到表面，从而产生一个空位，附近原子跳到这个空位上，就又产生一个新空位，这样空位可以逐步跳跃到晶体内部。
也可能由于热涨落晶体内部原子脱离格点，产生一个空位，这个原子可以经过多次跳跃，而跑到晶体表面的正常格点位置上，在晶体内形成空位。
## 反Schottky缺陷
与Schottky缺陷相对应的，还有一种*反Schottky缺陷*，也称为*间隙原子缺陷*。它是**晶体表面某一格点的原子移到晶体内部的间隙位置**。
显然，从晶体表面挤入拥挤的晶体内部形成反Schottky缺陷需要更大的能量，所以除小半径杂质原子外，一般不易单独形成此种缺陷。

|    特征    |    Frenkel缺陷     |   Schottky缺陷   |  反Schottky缺陷   |
| :------: | :--------------: | :------------: | :------------: |
| **缺陷组成** |    空位 + 间隙原子     |       空位       |       间隙       |
| **原子来去** |   晶格位置 → 间隙位置    |  晶格位置 → 晶体表面   |  晶体表面 → 间隙位置   |
| **晶体体积** |       基本不变       |       增加       |   基本不变或略微减小    |
| **晶体密度** |       基本不变       |       降低       |    升高或基本不变     |
| **电荷效应** | 空位与间隙原子电荷相反，整体中性 | 阴阳离子空位等量，整体中性  | 阴阳离子间隙等量，整体中性  |
| **常见性**  | 常见（尤其在阳离子小的晶体中）  | 非常常见（尤其在离子晶体中） |      极其罕见      |
| **主导因素** |  阳离子尺寸小，易于进入间隙   | 阴阳离子尺寸相近，结构开放  | 需要极高的形成能，通常不考虑 |
## 杂质原子
实际晶体中总是存在某些微量杂质。
杂质的来源一方面是在晶体生长过程中自然引入的，另一方面是为了改善晶体性质被人为引入的。
杂质原子在晶体中的占据方式有两种：
- 一种是杂质原子占据基质原子的位置，称为替位式杂质缺陷；
- 一种是杂质原子进人晶格原子间的间隙位置，称为填隙式杂质缺陷。

下图表示硅晶体中填隙式杂质和替位式杂质的示意图，图中$A$为间隙式杂质，$B$为替位式杂质。![[硅晶体中的间隙式杂质和替位式杂质.png]]
对于一定的晶体而言，杂质原子是形成替位式杂质还是形成间隙式杂质，这主要取决于杂质原子与基质原子几何尺寸的相对大小及其电负性。
