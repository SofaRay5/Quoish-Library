#光学 
# 理想光学系统
理想光学系统具有以下基本特性：
1. **点成点像**。即物空间的每一点，在像空间必有一个点与之对应，且只有一个点与之对应，这两个对应点称为物像空间的*共轭点*。
2. **线成线像**。即物空间的每一条直线在像空间必有一条直线与之对应，且只有一条直线与之对应，这两条对应直线称为物像空间的*共轭线*。
3. **平面成平面像**。即物空间的每一个平面，在像空间必有一个平面与之对应，且只有一个平面与之对应。这两个对应平面称为物像空间的*共轭面*。
4. **对称轴共轭**。即物空间和像空间存在着一对唯一的共轭对称轴。当物点$A$绕物空间的对称轴旋转一个任意角$\alpha$时，它的共轭像点$A’$也绕像空间的对称轴旋转同样的角度$\alpha$，*这样的一对共轭轴称为光轴*。

根据理想光学系统上述特征，可以得到如下推论：
- 物空间的任一个同心光束必对应于像空间中的一个同心光束；
- 若物空间中的两点与像空间中的两点共轭，则物空间两点的连线与像空间两点的连线也一定共轭；
- 若物空间任意一点位于一直线上，则该点在像空间的共轭点必位于该直线的共轭线上。

不过在均匀透明介质中，除平面反射镜以外，其他实际光学系统都不能绝对成完善像。
研究理想光学系统的意义在于，用它成的像可以作为衡量实际光学系统成像质量的标准。通常把理想光学系统计算公式（近轴光学公式)计算出来的像，称为实际光学系统的*理想像*。
另外，在设计实际光学系统时，会用理想像近似表示实际光学系统所成像的位置和大小，即实际光学系统设计的初始计算。
# 确定物象关系的条件
![[由两个共轭面确定物像关系.png]]
如图所示，在一个光学系统的近轴区，已知两个物面$\pi_{1}$和$\pi_{2}$的共轭像面分别为$\pi_{1}'$和$\pi_{2}'$，其垂轴放大率分别为$\beta_{1}$和$\beta_{2}$。
现考虑任一物点$A$，通过它的任一条入射光线与两个已知物面的交点分别为$B_{1}$和$B_{2}$，它们相对光轴的高度为$y_{1}$和$y_{2}$。$B_{1}$和$B_{2}$经过光学系统成像的像点$B_{1}'$和$B_{2}'$必然位于两个已知物面的共轭面$\pi_{1}'$和$\pi_{2}'$上，其相对光轴的高度分别等于$\beta_{1}y_{1}$和$\beta_{2}y_{2}$，从而可以确定像点$B_{1}'$和$B_{2}'$的位置，并且经过$B_{1}'$和$B_{2}'$的光线就是入射光线对应的出射光线。
通过$A$点再作另外一条光路，它与两个已知物面的交点为$C_{1}$和$C_{2}$；同理，可以作出相应的出射光线，它与$\pi_{1}'$和$\pi_{2}'$的交点为$C_{1}'$和$C_{2}'$。显然，两条出射光线$B_{1}'B_{2}'$和$C_{1}'C_{2}'$的交点就是$A$点的像点$A’$。
由此可见，当光学系统成完善像时，**已知两个共轭面及其垂轴放大率**就可以完全确定光学系统的成像关系。
![[由一对共轭面和两对共轭点确定物像关系.png]]
如图所示，另一个光学系统的近轴区，已知一个物面$\pi$和其共轭像面$\pi'$，它们之间的垂轴放大率为$\beta$，还已知两对共轭点，即$B_{0},C_{0}$和其高斯像点$B_{0}'C_{0}'$。
现考虑任一物点$A$，作出通过它和$B_{0}$的一条入射光线，该入射光线与已知的物面$\pi$的交点为$B$，根据$\pi$的共轭像面$\pi'$的位置和它们之间的垂轴放大率可以确定$B_{1}$的像点$B_{1}'$则由$B_{0}'$和$B_{1}'$就确定了出射光线。
同理，可以确定出与通过$A$和$C_{0}$的入射光线相应的出射光线$C_{0}'C_{1}'$。这两条出射光线$B_{0}'B_{1}'$和$C_{0}'C_{1}'$的交点即为$A$的像点$A’$。
由此可见，当光学系统成完善像时，**已知一对共轭面及其垂轴放大率和两对共轭点**，也可以完全确定光学系统的成像关系。
# 基点和基面
主点、焦点、节点称为理想光学系统的*基点*。
主平面、焦平面、节平面称为理想光学系统的*基面*。
通常给出一对主点、一对焦点、一对主平面，这样就能确定理想光学系统的物像关系。
## 主点和主平面
在理想光学系统中，将垂轴放大率$\beta=1$的一对共轭面称为*主平面*，其中物面称为*物方主平面*，像面称为*像方主平面*。
物方主平面和光轴的交点称为*物方主点*，用$H$表示；像方主平面和光轴的交点称为*像方主点*，用$H’$表示。
主平面具有以下的性质：
假定物空间的任意一条光线和物方主平面的交点为$P$，它的共轭光线与像方主平面交于$P’$点，则$P$和$P’$距光轴的距离相等，$PP'$和。![[主平面.png]]
## 焦点和焦平面
在理想光学系统中定义了两对特殊的共轭点。
一对共轭点为光轴上位于无穷远的物点和其像点$F’$，$F’$称为*像方焦点*，如下图$(a)$所示。像方焦点相对像方主点沿光轴的线度称为*像方焦距*，表示为$f’$。经过像方焦点和光轴垂直的平面称为*像方焦平面*。
另外一对共轭点为光轴上位于无穷远的像点和其物点$F$，$F$称为*物方焦点*，如下图$(b)$所示。物方焦点相对物方主点沿光轴的线度称为*物方焦距*，表示为$f$。经过物方焦点和光轴垂直的平面称为*物方焦平面*。![[焦平面.png]]
## 节点和节平面
焦点是按照共轭点的空间位置定义的，有时根据光学系统对光线的传播方向的影响定义共轭点，这就是节点。
在理想光学系统中，将角放大率$\alpha=1$的一对共轭面称为*节平面*，其中物面称为*物方节平面*，像面称为*像方节平面*。
物方主平面和光轴的交点称为*物方节点*，用$J$表示；像方主平面和光轴的交点称为*像方节点*，用$J’$表示。
节点具有以下的性质：
通过物方节点的光线经过光学系统后，出射光线必然通过像方节点，并且光的传播方向不变，如图所示，恒有$U=U’$。![[节平面.png]]
# 基本光学元件的基点和基面
## 薄透镜
薄透镜的物方主点和像方主点重合，位于$l=l'=0$光心处。
薄透镜的物方节点和像方节点重合，位于$l=l'=0$光心处。
## 折射球面
折射球面的物方主点和像方主点重合，位于$l_{H}=l_{H}'=0$顶点处。
折射球面的物方节点和像方节点重合，位于$l_{J}=l_{J}'=r$球心点处。
## 反射球面
反射球面的物方主点和像方主点重合，位于$l_{H}=l_{H}'=0$顶点处。
反射球面的物方节点和像方节点重合，位于$l_{J}=l_{J}'=r$球心点处。
# 理想光学系统的物像关系
## 图解法求像
对于理想光学系统，只要知道基点的位置，其成像性质就确定了，可以根据基点的性质通过作图方法确定物体和其像的位置和大小，这种方法称为图解法。
![[理想光学系统图解法求像.png]]
如图所示，理想光学系统的主点和焦点位置已知时，欲求一垂轴物体$AB$经光学系统的像，只需过$B$点作两条特殊的入射光线：
- 其中一条光线平行于光轴，出射光线必过像方焦点$F’$；
- 另一条光线过物方焦点，出射光线必平行于光轴。

两出射光线的交点$B’$就是物点$B$的像点。因$AB$垂直于光轴，故过像点$B’$作垂直于光轴的线段$A'B'$就是物体$AB$经系统后所成的像。![[利用像方焦平面求入射光线的出射光线.png]]也可以认为入射光线是从物方焦平面$FB$上发出的一条，这时过入射光线$AB$与物方焦平面$B$的交点作一条平行于光轴的辅助线$BN$，其出射线必过像方焦点$F'$。由于入射光线的出射线$N'F'$平行于辅助光线的出射线，因而可求得任意光线的出射线$M'A'$。

$$x x'=f f' $$
### Gauss公式
$$\frac{f}{l}+\frac{f'}{l'}=1$$
### 焦距关系
$$\frac{f'}{f}=- \frac{n'}{n}$$对于同一介质$n=n'$，则$$\frac{f'}{f}=-1$$
### 光焦度
$$\varphi=\frac{n'}{f'}=-\frac{n}{f}$$
### Lagrange-Helmholtz不变量
$$ny\tan u=n'y'\tan u'$$

### 垂轴放大率
$$\beta=\frac{l'/n'}{l/n}$$
### 轴向放大率
$$\alpha=\frac{l'^2/n'}{l^2/n}$$
### 角放大率
$$\gamma=\frac{l}{l'}$$
### 放大率关系
$$\alpha \gamma=\beta$$

# 光学系统基点的测量
## 节点测量法
*节点测量法*，也称为节点架法或转台法，是一种非常精确的测量方法，尤其适用于厚透镜、透镜组或复杂光学系统的焦距测量。
它的核心原理是利用当光学系统绕其像方节点$N'$旋转时，如果入射光线平行于光轴（即来自无穷远的物点），那么其像点位置不会发生横向移动。
**步骤：**
1. **粗调**：将光学系统（透镜）安装在节点架上，大致调节其高度和方向，使光轴与导轨平行。
2. **寻找像点**：用测微目镜找到平行光管十字丝的最清晰像。
3. **旋转测试**：轻微转动节点架。观察测微目镜中的像：
    - 如果像发生横向移动，说明转轴不在节点上。
    - 当像静止不动时，节点架的转轴就恰好通过了光学系统的像方节点$N$。
4. **定位节点**：记录下此时转轴在光具座标尺上的位置，这个位置就是像方节点$N’$的位置。
5. **测量焦距**：保持节点架不动，移动测微目镜，直到看到平行光管十字丝最清晰的像。此时，测微目镜的位置就是像方焦点$F'$的位置。像方节点$N’$到像方焦点$F'$的距离，即为光学系统的焦距$f$。  
## 二次成像法
*二次成像法*，也称为共轭法、贝塞尔法或位移法。它特别适合测量凸透镜的焦距，其最大优点是不需要精确知道透镜的光心位置。
二次成像法利用了一个重要的现象：当物与屏的间距$D > 4f$时，移动透镜可以在屏上两次接收到清晰的实像——一次是放大的清晰实像，一次是缩小的清晰实像，这两个位置是共轭的。
设透镜两次成像位置之间的距离为$d$。
在L1位置：$u_{1} + v_{1} = D$；在L2位置：$u_{2} = v_{1}, v_{2} = u_{1}$；
解出：$u_{1} = (D - d)/2,\quad v_{1} = (D + d)/2$。
代入Gauss公式，就能得到$f = (D² - d²) / (4D)$
**步骤：**
1. **粗测焦距**：用粗略方法（如太阳光法）预估焦距$f$，以确保后续$D > 4f$。
2. **布置光路**：将物、透镜、像屏按顺序放在光具座上，调节共轴。
3. **固定物像距**：将物屏和像屏固定，并测量它们之间的距离$D$。确保$D > 4f$。
4. **寻找清晰像**：将透镜从靠近物屏的一端缓慢移向像屏，记录下第一次出现清晰放大像时透镜的位置$L_{1}$。
5. **再次寻找清晰像**：继续移动透镜，记录下第二次出现清晰缩小像时透镜的位置$L_{2}$。
6. **计算**：测量$d = |L1 - L2|$，代入公式$f = (D² - d²) / (4D)$即可计算出焦距。
