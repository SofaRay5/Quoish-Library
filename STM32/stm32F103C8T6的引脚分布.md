# stm32F103C8T6的引脚分布
![[stm32f103c8t6引脚分布.png]]
彩色引脚是用户不可通过编程操控的引脚
灰色引脚是用户可以通过编程操控的引脚
- 备用电池引脚：`VBAT`即V Battery，用于断电时给单片机的备份域（Backup Domain）供电通常使用一枚纽扣电池
- 复位引脚：`NRST`即Non-Maskable Reset（非屏蔽复位引脚），通常连接按钮，按下后芯片复位
- 启动引脚：`BOOT`即启动引脚（boot v. 启动），用于选择芯片的启动模式
- GPIO引脚：剩下的可编程引脚都是GPIO（General Purpose Input Output，通用输入输出）引脚分为GPIOA、GPIOB、GPIOC、GPIOD等几类，GPIOA包括PA0、PA1、PA2……PA15；GPIOB包括PB0、PB1、PB2……PB15；以此类推
